# 启程录

> 每一次投递，都是新的启程

一个轻量级的求职投递跟踪工具，专为秋招/春招/暑期实习设计。

## 功能

- 📋 **看板视图** — 按投递状态分列展示，一目了然
- 📃 **列表筛选** — 按企业类型、城市、求职季快速筛选
- 📅 **面试日历** — 月历视图展示所有面试安排
- 📝 **面经本** — 所有面经汇总，支持原地编辑
- 📊 **数据统计** — 饼图、进度条可视化投递情况
- ✨ **AI 助手** — 面经搜索、面试复盘、智能周报（支持 DeepSeek / 通义千问）
- 🏆 **成就系统** — 投递、面试、Offer 自动解锁星级徽章
- 💾 **数据安全** — JSON 导入导出，版本兼容，备份提醒

## 使用

直接打开 `index.html` 即可使用，或访问在线版本。

所有数据存储在浏览器本地，建议定期导出备份。

## 技术

纯前端，单 HTML 文件，零依赖，无需服务器。
