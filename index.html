📖 使用说明

📋 看板

所有投递按状态分列显示，从"待投递"到"Offer/被挂/已拒"，左右滑动查看所有列。顶部可按求职季（暑期实习/秋招/春招）切换筛选。

📃 列表

全部投递的列表视图，支持搜索和按企业类型/城市快速筛选。点击任意记录可编辑详情。

📅 日历

以月历形式展示所有面试安排。面试日期会在日历上标点，点击查看当天详情。在每条投递里添加面试记录时填写日期即可。

📝 面经

自动汇总所有面试轮次中写过的面经笔记，按时间倒序排列，支持搜索。不用一个个点进投递记录去翻了。

📊 统计

可视化查看投递进度分布、企业类型饼图、城市分布饼图，以及各求职季投递量对比。

➕ 新增投递

点右下角 + 按钮新增。只需填公司名就能保存，其余字段按需填写。每条投递可以添加多轮面试记录（含面经和该轮结果），并可记录投递链接方便跳转官网查看进度。

✨ AI 助手

点右上角 ✨ 打开。包含三个功能：面经搜索（输入公司+岗位，AI 帮你分析常见面试问题和备战要点）、面试复盘（选择一条面经，AI 分析你的薄弱点并给改进建议）、智能周报（AI 根据你的全部数据生成求职进度总结和行动建议）。

使用 AI 功能需要在 ⚙️ 设置中选择服务商（DeepSeek 或 通义千问）并填入对应的 API Key。Key 仅存在你的浏览器里，不会泄露。

🏆 成就

点右上角 🏆 查看。系统会根据你的投递和面试数据自动解锁成就徽章，给枯燥的求职旅程加点乐趣。

⚙️ 账户与数据

注册登录后，数据自动同步到云端，换设备也不怕丢。未登录时以离线模式使用，数据仅存在当前浏览器。设置中可导出/导入 JSON 备份、管理账户。

即使登录了账户，仍建议偶尔导出 JSON 备份以防万一。
