📖 使用说明

看板

投递按状态从上到下分流水线排列。点击顶部统计标签可快速筛选，比如只看"一面中"的记录。每个阶段可展开/折叠。

列表

全部投递的列表视图，支持搜索和快速筛选。

日历

月历形式展示面试安排。在投递详情中添加面试记录并填写日期即可显示。

面经本

自动汇总所有面试面经笔记，支持搜索和直接编辑。

新增投递

点 + 按钮新增。添加面试记录时可设置日期和具体时间，方便在日历中查看。勾选面试结果为"通过"或"挂了"后，投递状态会自动推进（如一面通过 → 自动变为二面状态），无需手动切换。

数据备份

登录后数据自动同步到云端，换设备也不丢。设置中也可导出/导入 JSON 备份。

虽然数据已云同步，仍建议偶尔导出 JSON 备份以防万一。
